# My Porfolio Website

This project is a sample of my React understanding and is the code behind my portfolio website

## Install

Install dependencies either using

```
yarn
```

or

```
npm install
```

# Running the project

This is a vite project with React JS.
<br/>
Run it on dev server

```
npm run dev
```

# Preview Live
You can also preview it live using

```
npm run build
```
This will build the project on distribution folder. Run it by using

```
npm run preview
```
